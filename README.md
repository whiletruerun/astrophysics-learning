# astrophysics-learning

This repo has small projects im using to learn computational astrophysics and cosmology

I'm currently focusing on learning how numerical and computational methods can be used to observe and analyse datasets

## Projects

### 1. Orbit Simulation (Newton's law of gravitation and Euler's method)
- Simulated orbital motion of planet using Newton's law of gravitation and Euler's method
- Explored basic numerical integration methods
- Visualized paths taken by planets under the influence of gravity

### 2. Light curve analysis (Lightkurve)
- Worked with the lightkurve library on python to plot a basic flux vs time curve
- Explored how transits can be used to detect and study new planets

## Tools used
- Python
- Google Colab
- Numpy
- Matplotlib
- Lightkurve

This is a learning repo, so all the projects are basic and beginner-level as I build my understanding
